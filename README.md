# Ram Ratan & Co. - Official Website

Welcome to the official website repository for **Ram Ratan & Co.**, a pioneer in manufacturing specialty materials for agricultural research and seed testing.

## Project Overview

This is a responsive, single-page website built to showcase the company's products, services, and client portfolio. It includes interactive elements like a hero carousel, product grids, and a contact form modal.

## Features

-   **Hero Carousel**: Dynamic slideshow displaying key offerings and services using local imagery.
-   **Product Showcase**: Grid layout highlighting "Specialty Research Materials" with high-quality images.
-   **Services Section**: Overview of Liaison, Import/Export, and Consultancy services.
-   **Client Portfolio**: Section displaying logos of trusted clients (e.g., Sungro Seeds, Mahyco).
-   **About Us**: Company history and mission statement with a dedicated verification logo.
-   **Contact Modal**: Interactive "Get in Touch" form that opens in a popup dialog.
-   **Responsive Design**: Optimized for desktop and mobile viewing with a mobile-friendly hamburger menu.

## Technologies Used

-   **HTML5**: Semantic structure.
-   **CSS3**: Custom styling, Flexbox/Grid layouts, CSS transitions (no external CSS frameworks used).
-   **JavaScript**: Logic for the image carousel, mobile menu toggle, and contact modal.
-   **Font Awesome**: Icons for UI elements.

## Setup Instructions

1.  Clone or download the repository.
2.  Open `index.html` in any modern web browser.
3.  No build step or server is required for basic viewing (static site).

## File Structure

-   `index.html`: Main HTML file.
-   `style.css`: All custom styles and responsive rules.
-   `script.js`: Functionality for carousel and modal.
-   `images/`: Directory containing all visual assets (product photos, logos, carousel slides).
